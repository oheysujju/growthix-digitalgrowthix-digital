# Growthix Digital Website

Modern single-page website for Growthix Digital - Digital Marketing Agency based in Indore.

## Features
- Blue & White premium theme
- Fully responsive
- Smooth animations
- Services showcase (Social Media, Website Design, SEO, etc.)
- Pricing packages
- Contact form
- Mobile-friendly

## How to deploy on GitHub Pages

1. Create a new GitHub repository
2. Upload `index.html` and `README.md`
3. Go to Settings → Pages → Select main branch → Save
4. Your site will be live at `https://yourusername.github.io/repo-name`
